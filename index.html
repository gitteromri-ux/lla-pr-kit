3 · Steps + paste-ready article (per outlet)

1) PR.com

Create account Sign in
Length: no hard limit · paste the FULL body (~355 words / ~2,300 chars) ✓
  1. Create account → solve reCAPTCHA → Sign Up.
  2. Sign in → "Submit Press Release".
  3. Paste fields → choose the Free package (look for $0/basic) → submit.

2) PRLog

Create account
Length: no hard limit · paste the FULL body (~355 words) ✓
Submit page
  1. Create free account → solve image CAPTCHA.
  2. Dashboard → "Submit Press Release" → paste → submit. Usually same-day live.

3) openPR — use company email, not Gmail

Open submit form
Requires min ~1,700 chars · FULL body (~2,300 chars) ✓ qualifies
  1. Public form, no account. Enter omri.gitter@eteachergroup.com.
  2. Fill fields + full postal address + optional image → CAPTCHA → accept terms → Preview → Confirm.
  3. Limit: 1 free / 30 days.

4) IssueWire

Sign up
Limit: 250–2,000 words · FULL body (~355 words) ✓ fits
  1. Sign up → look for "FIRST PR IS FREE NOW" → reCAPTCHA → Get Started (free = $0.00).
  2. Create → Optimize → Submit. Body 250–2000 words. Editor review before live.

5) 1888PressRelease — uses the SHORT body (~300 words)

Register Login
Keep ~300 words → paste the SHORT body below ⚠ (full body is too long here)
  1. Create free account → reCAPTCHA → Submit.
  2. Login → "Submit Press Release" → paste the SHORT version (no ALL-CAPS headline) → submit for review.

6) PRFree

Register Login
Length: no hard limit · paste the FULL body (~355 words) ✓
  1. Register → accept terms → Cloudflare Turnstile → Create.
  2. Login → submission tool → choose Free plan → submit. Instant publish.

7) PressReleasePoint

Open site
Length: no hard limit · paste the FULL body (~355 words) ✓
  1. Click "Submit Free Press Release" → register/login → form → paste → CAPTCHA → submit.
  2. Limit: 1 free / 3 months.

8) Free-Press-Release.com

Open site
Limit: ≤3,000 chars · FULL body (~2,300 chars) ✓ fits
  1. Register free account → Submit → headline, summary, body (≤3,000 chars basic), categories, media → submit.
